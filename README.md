# 🛒 Amazon Homepage Clone

A fully responsive Amazon homepage clone built using HTML, CSS, and JavaScript.

## 🚀 Features

- Amazon-style Navigation Bar
- Search Bar UI
- Hero Section
- 8 Product Category Boxes
- Responsive Grid Layout
- Smooth "Back to Top" Scroll
- Clean UI and structured CSS

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

## 📱 Responsive Design

- 4 columns (Desktop)
- 3 columns (Tablet)
- 2 columns (Mobile)
